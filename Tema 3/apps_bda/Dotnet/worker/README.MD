# .NET for Apache Spark Setup Guide

This project requires **.NET for Apache Spark** to be installed on your system.

## Download the Required Binary

Before running the application, you **must download the .NET for Apache Spark binary** from the official GitHub releases page:

**https://github.com/dotnet/spark/releases/**

Download the binary that matches:

- Your operating system (Windows / Linux / macOS)
- Your Spark version
- Your .NET version
