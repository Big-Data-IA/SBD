# Apache Spark + .NET for Apache Spark Setup

This project requires a working installation of **Apache Spark** and **.NET for Apache Spark**.

Please follow the instructions below before running the project.

---

## Important Note for C# Developers (.NET for Apache Spark)

If you are working in **C#**, ensure the Spark version you install is **compatible with .NET for Apache Spark**.

Check the compatibility matrix here:
[https://github.com/dotnet/spark/releases/](https://github.com/dotnet/spark/releases/)

> ⚠️ **.NET for Apache Spark does NOT support all Spark versions.**
> Make sure the Spark version you install matches the version required by the .NET for Apache Spark binary.

## 2. Install .NET for Apache Spark Binary

Download the matching **.NET for Apache Spark** release from:

➡️ **[https://github.com/dotnet/spark/releases/](https://github.com/dotnet/spark/releases/)**

Choose the binary that corresponds to:

- Your OS
- Your Spark version
- Your .NET version

Extract it and ensure the binaries are available in your `PATH`.
